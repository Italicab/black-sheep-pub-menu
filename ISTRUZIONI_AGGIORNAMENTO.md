# Aggiornamento menu reale - Black Sheep Pub

Questa versione contiene il menu reale in italiano e inglese, modificabile da admin.html.

Per aggiornare il sito online:
1. Estrai questo ZIP.
2. Carica tutti i file nel repository GitHub.
3. Fai commit.
4. Vercel aggiornerà il sito automaticamente.

Puoi poi correggere menu del giorno, piatti, prezzi e traduzioni da /admin.html.
